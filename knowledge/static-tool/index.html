Static Tool Holders (非動力刀座)

What is a Static Tool Holder?

A static (non-driven) tool holder mounts fixed cutting tools on the CNC lathe turret — turning tools, boring bars, grooving tools, threading tools. The workpiece rotates; the tool is stationary.

Common Types

  • OD turning holders: External turning, facing, chamfering
  • Boring bar holders: Internal turning, ID boring
  • Grooving / parting holders: Groove cutting, cut-off
  • Threading holders: External and internal thread turning
  • Floating reamer holders (浮動絞刀柄): Self-centering for precision ID finishing

Interface Standards

  • VDI (DIN 69880): Cylindrical shank. VDI 20, 25, 30, 40 sizes.
  • BMT (Base Mount Turret): Bolt-on face mount. Higher rigidity.
  • Machine-specific: TSUGAMI, Miyano, STAR have proprietary turret designs.

Other Hardware (五金)

  • Knurling tools (滾花刀): Straight, diagonal, cross-knurl patterns
  • Form tools (成型刀): Custom profile tools for special contours
  • Punch tools (沖孔刀): Marking, center punching on lathe
  • Insert shims (刀片墊片): Precision height adjustment for turning inserts

Key Selection Parameters

Machine brand/model, turret type (VDI/BMT/proprietary), shank size, insert pocket type, coolant method, cutting direction.
繁體中文

什麼是非動力刀座？

非動力刀座將固定式刀具安裝於 CNC 車床刀塔上，包括車刀、搪孔刀、切槽刀、車牙刀等。工件旋轉，刀具靜止。

常見類型

  • 外徑車刀座：外徑車削、端面車削、倒角
  • 搪孔刀座：內徑車削、內孔搪孔
  • 切槽 / 切斷刀座：切槽、切斷
  • 車牙刀座：外牙及內牙車削
  • 浮動絞刀柄：自動定心，用於精密內孔加工

介面標準

  • VDI（DIN 69880）：圓柱柄，VDI 20、25、30、40 尺寸。
  • BMT（Base Mount Turret）：螺栓鎖付端面安裝，剛性更高。
  • 機台專用：TSUGAMI、Miyano、STAR 有各自的刀塔設計。

其他五金

  • 滾花刀：直紋、斜紋、菱形滾花
  • 成型刀：特殊輪廓客製刀具
  • 沖孔刀：車床上的標記、打中心
  • 刀片墊片：車刀片精密高度調整

選型關鍵參數

機台品牌/型號、刀塔類型（VDI/BMT/專用）、柄部尺寸、刀片槽型式、冷卻方式、切削方向。
한국어

정적 툴 홀더란?

정적(비구동) 툴 홀더는 CNC 선반 터렛에 고정 절삭 공구를 장착합니다 — 선삭 공구, 보링 바, 홈 파기 공구, 나사 가공 공구. 공작물이 회전하고 공구는 정지합니다.

일반 유형

  • 외경 선삭 홀더: 외경 선삭, 단면 가공, 챔퍼링
  • 보링 바 홀더: 내경 선삭, 내경 보링
  • 홈 파기 / 절단 홀더: 홈 가공, 절단
  • 나사 가공 홀더: 외경 및 내경 나사 가공
  • 플로팅 리머 홀더: 자동 센터링. 정밀 내경 마무리 가공용.

인터페이스 규격

  • VDI (DIN 69880): 원통형 섕크. VDI 20, 25, 30, 40 크기.
  • BMT (Base Mount Turret): 볼트온 정면 장착. 더 높은 강성.
  • 기계 전용: TSUGAMI, Miyano, STAR는 고유 터렛 설계 보유.

기타 하드웨어 (五金)

  • 널링 공구 (滾花刀): 직선, 대각선, 크로스 널링 패턴
  • 성형 공구 (成型刀): 특수 윤곽용 맞춤 공구
  • 펀치 공구 (沖孔刀): 선반에서 마킹, 센터 펀칭
  • 인서트 심 (刀片墊片): 선삭 인서트의 정밀 높이 조정

주요 선정 파라미터

기계 브랜드/모델, 터렛 유형 (VDI/BMT/전용), 섕크 크기, 인서트 포켓 유형, 절삭유 방식, 절삭 방향.
